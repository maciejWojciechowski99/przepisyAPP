PrzepisyAPP to  aplikacja webowa, która umożliwia użytkownikom dodawanie, przeglądanie i zarządzanie przepisami kulinarnymi. Aplikacja została stworzona w języku PHP z wykorzystaniem MySQL jako bazy danych.

Podstawowe funkcjonalności

1. Dodawanie Przepisów
Użytkownicy mogą dodawać nowe przepisy, podając tytuł, opis oraz szczegóły przepisu. Dodane przepisy są przechowywane w bazie danych.

2. Przeglądanie Przepisów
Zarejestrowani użytkownicy mogą przeglądać dodane przepisy, zarówno swoje.

3. Edycja Przepisów
Użytkownicy mają możliwość edycji swoich dodanych przepisów, zmieniając tytuł, opis.

4. Usuwanie Przepisów
Użytkownicy mogą usuwać swoje przepisy z bazy danych.

5. Ulubione Przepisy
Użytkownicy mogą dodawać przepisy do ulubionych, co umożliwia łatwe odnalezienie ulubionych dań.

6. Rejestracja i Logowanie
Aplikacja obsługuje system rejestracji i logowania. Użytkownicy muszą być zalogowani, aby dodawać, edytować i usuwać przepisy.

7. Logi Zdarzeń
Aplikacja rejestruje ważne zdarzenia, takie jak dodawanie, edycja, usuwanie przepisów oraz logowanie i wylogowanie użytkowników. Logi są przechowywane w bazie danych.

8. Lista Todo

Aplikacja posiada listę todo.

9. Losowanie Przepisu

Aplikacja oferuje funkcję losowania przepisu. Użytkownicy, którzy mają trudność z wyborem, co gotować, mogą skorzystać z opcji losowego przepisu, aby uzyskać inspirację.

10. Technologie

Aplikacja została stworzona przy użyciu PHP, MySQL oraz JavaScript.


Wymagania Systemowe
- Serwer WWW obsługujący PHP (np. Apache)
- Baza danych MySQL
- PHP 7.x

Instalacja
1. utwórz bazę "przepisy" i zaimportuj strukturę bazy danych z pliku "database.sql".
2. Uruchom aplikację na serwerze WWW.
